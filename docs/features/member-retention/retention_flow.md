# 잔류 부원 이월 흐름

## 목적

운영진이 종료된 이전 학기의 부원 중 잔류한다고 응답한 사람을 진행 중인 새 학기로 옮긴다.
DB에 바로 저장하지 않고 먼저 결과를 보여준 뒤, 운영진이 확정한 사람만 저장한다.

## 지원하는 원본

- 표 파일(CSV)
- 엑셀 파일(XLSX): 여러 시트 중 하나를 선택할 수 있다.
- Google Form 응답이 연결된 Google Sheet
- 일반 Google Sheet

Google Form 응답 Sheet와 일반 Google Sheet는 행과 열을 읽는 방식이 같다. PDF, 이미지,
일반 Google Docs는 표 구조를 안정적으로 판별하기 어려워 지원하지 않는다.

## 입력 규칙

원본 첫 행을 열 제목으로 사용한다. 화면에서 원본 열과 다음 필드를 연결한다.

| CrewCat 필드 | 필수 | 설명 |
|---|---:|---|
| 이메일 | 예 | 이전 학기 부원과 같은 사람인지 찾는 기준 |
| 잔류 여부 | 예 | `잔류`, `예`, `Y` 등 운영진이 지정한 값을 잔류로 판단 |
| 이름 | 아니오 | 미리보기 확인용이며 기존 `Person` 정보를 덮어쓰지 않음 |
| 학번 | 아니오 | 미리보기 확인용이며 기존 `Person` 정보를 덮어쓰지 않음 |

- 이메일은 앞뒤 공백을 제거하고 소문자로 바꾼 뒤 비교한다.
- 잔류로 인정할 값은 화면에서 쉼표로 구분해 입력한다.
- 지정한 값과 다른 잔류 응답은 `잔류 안 함`으로 제외한다.
- 이름과 학번은 잔류 부원 이월 시 새 인물을 만드는 데 쓰지 않는다. 이전 학기에 이미 존재하는
  `Person`만 새 학기에서 재사용한다.

## 행별 판정

| 판정 | 의미 | 저장 여부 |
|---|---|---:|
| `READY` | 이전 학기 부원이며 새 학기에 아직 없음 | 저장 대상 |
| `NOT_RETAINED` | 잔류로 인정한 값이 아님 | 제외 |
| `INVALID` | 이메일이 비었거나 형식이 잘못됨 | 제외 |
| `DUPLICATE_IN_SOURCE` | 원본 안에 같은 이메일이 둘 이상 있음 | 같은 이메일 행을 모두 제외 |
| `NOT_PREVIOUS_MEMBER` | 이전 학기 부원 명단에 없음 | 제외 |
| `ALREADY_TARGET_MEMBER` | 새 학기에 이미 등록됨 | 제외 |

원본 중복은 어느 행이 맞는지 프로그램이 임의로 선택하지 않고 같은 이메일 행을 모두 제외한다.
운영진이 원본을 고친 뒤 다시 미리보기할 수 있다.

## 실행 순서

```text
종료된 이전 학기 + 진행 중인 새 학기 선택
  → 표 파일(CSV·엑셀) 업로드 또는 Google Sheet 불러오기
  → 사용할 시트 선택
  → 이메일·잔류 여부 열 연결
  → 미리보기 요청
  → 서버가 이전/새 학기 부원과 중복을 다시 검사
  → 운영진이 READY 행 확정
  → 서버가 조건을 다시 검사한 뒤 GenerationMember 생성
```

미리보기는 DB를 변경하지 않는다. 확정 요청에서도 서버가 다시 검사하므로, 미리보기 이후 다른
운영진이 먼저 이월했더라도 같은 부원이 두 번 생성되지 않는다.

## 저장 규칙

- 새 `GenerationMember`는 `joined_source=RETENTION`, `status=ACTIVE`로 생성한다.
- 기존 `Person`을 그대로 참조하며 이름·학번·연락처를 수정하지 않는다.
- DB의 `UNIQUE(generation_id, person_id)`가 같은 학기의 마지막 중복 방지선이다.
- 같은 확정 요청을 다시 보내면 이미 등록된 부원은 건너뛰고 결과 건수에 표시한다.
- 이전 학기의 `GenerationMember`는 수정하거나 삭제하지 않는다.

## Google 권한

관리 화면 로그인과 Sheet 읽기 권한은 서로 분리한다. 운영진이 Google Sheet 기능을 처음 사용할 때
`spreadsheets.readonly` 범위로 별도 동의한다. 접근·갱신 토큰은 서버 DB에 암호화해 저장하며 브라우저로
전달하지 않는다. 원본 Sheet 내용과 업로드 파일 자체는 저장하지 않는다.

## 제한

- 표 파일은 CSV(`.csv`)와 엑셀(`.xlsx`)만 허용하고 요청 크기는 최대 5MB다.
- 한 시트는 열 100개, 데이터 행 2,000개까지 읽는다.
- Google Sheet 자동 동기화, 가져오기 이력, 열 연결 설정 저장은 후속 범위다.
